# Customer Review Classification using BERT

##  Project Overview
This project fine-tunes a BERT (Bidirectional Encoder Representations from Transformers) model to classify customer reviews into meaningful categories such as:

- Feature Requests  
- Navigation Issues  
- Bug Reports  
- General Feedback  

The goal is to automatically understand user responses and extract actionable insights from customer reviews.

---

## Model Details
- Model: BERT (fine-tuned transformer model)
- Task: Multi-class Text Classification
- Dataset: Synthetic dataset generated using Gemini
- Evaluation Accuracy: **85%**

---

##  Results
The fine-tuned model achieved **85% accuracy** on the evaluation dataset, demonstrating strong performance in understanding and categorizing customer feedback.
